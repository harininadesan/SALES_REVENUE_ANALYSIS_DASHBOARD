# SALES_REVENUE_ANALYSIS_DASHBOARD

## Project Overview
This project is a Power BI dashboard developed to analyze sales and revenue data. It provides interactive visualizations and key performance indicators (KPIs) to help understand business performance.

## Features
- Total Sales KPI
- Total Profit KPI
- Total Quantity Sold KPI
- Monthly Sales Trend
- Sales by Category
- Profit by Product
- Sales by Region
- Interactive Region and Category Filters (Slicers)

## Tools Used
- Microsoft Power BI
- CSV Dataset
